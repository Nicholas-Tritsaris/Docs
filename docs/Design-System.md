## Overview
The Design-System repository is a centralized design system with Y2K/retro aesthetics, built from extracted components across multiple projects. It provides a consistent and reusable set of UI components, layouts, and styles to help developers create cohesive and visually appealing applications.

## Features
The Design-System repository includes the following key features:
* **6 Core Components**: Button, Card, Input, Badge, Navbar, and Marquee, which provide a solid foundation for building user interfaces.
* **3 Layout Templates**: Retro, Terminal, and Dashboard, which offer pre-designed layouts for common use cases.
* **4 Style Modules**: Colors, Typography, Spacing, and Effects, which enable developers to customize the visual appearance of their applications.
* **Retro Storybook**: An interactive component preview with code snippets, allowing developers to explore and test components in a sandbox environment.
* **Pure HTML/CSS/JS**: The design system is built using only HTML, CSS, and JavaScript, making it framework-agnostic and easy to integrate into existing projects.
* **GitHub Pages Compatible**: The repository is ready to deploy to GitHub Pages, providing a simple way to host and share the design system.

## Installation
To get started with the Design-System repository, follow these steps:
1. Link the stylesheet by adding the following line of code to your HTML file: `<link rel="stylesheet" href="Design-System/styles/main.css">`
2. Use the provided components by adding the corresponding HTML code to your project. For example:
```html
<!-- Button -->
<button class="btn btn-default">Click Me</button>

<!-- Card -->
<div class="card card-default">
  <div class="card-header">
    <h3 class="card-title">Title</h3>
  </div>
  <div class="card-body">Content</div>
</div>

<!-- Input -->
<input type="text" class="input input-default" placeholder="Enter text...">
```

## Project Structure
The Design-System repository is organized into the following directories:
* `styles/`: Contains the main stylesheet (`main.css`) and individual style modules for colors, typography, spacing, and effects.
* `components/`: Includes the core components, each with its own directory and HTML file (e.g., `button/`, `card/`, `input/`, etc.).
* `layouts/`: Provides pre-designed layout templates for common use cases.
