Rare is an open-source alternative to the Epic Games Launcher, utilizing the Legendary command-line tool. It provides a graphical interface written in PySide6, offering a native and lightweight experience across major platforms. Rare aims to be a feature-full and seamless integration with Legendary, allowing users to enjoy their games without unnecessary distractions.

## Overview
Rare is designed to be a user-friendly and efficient solution for managing Epic Games Launcher functionality. By leveraging the Legendary command-line tool, Rare provides a graphical interface that is both intuitive and customizable. The project focuses on delivering a native and lightweight experience, ensuring that Rare runs smoothly on various platforms.

## Features
The key features of Rare include:
* **Native support** for major platforms, ensuring a seamless experience across different operating systems
* A **lightweight design**, allowing Rare to run efficiently without consuming excessive system resources
* **Seamless integration** with Legendary, enabling a cohesive and feature-rich experience
* **Customization options**, providing users with the ability to personalize their Rare setup
* **Package management**, offering a convenient way to manage and organize game libraries

## Installation
Rare provides a straightforward installation process, with platform-specific instructions available.
### Windows
An `.msi` installer is available for Windows users, making it easy to install and set up Rare on their systems.
For more information on installation and usage, please refer to the [official Rare repository](https://github.com/RareDevs/Rare) or join the [Rare Discord community](https://discord.gg/YvmABK9YSk) for support and discussion.
