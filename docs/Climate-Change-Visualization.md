Climate Change Visualization is a *WebGL-globe* project that aims to visualize surface temperature anomalies over the past century. Inspired by *Google Chrome Experiments*, this project utilizes data from **NASA GISS Surface Temperature Analysis (GISTEMP v4)** to create an interactive and informative visualization of temperature changes. The project's website layout is adapted from *Cathal Mc Daid's* adaptive mobile spam globe, providing a user-friendly interface for exploring the data.

## Overview
The Climate Change Visualization project is designed to provide a clear and concise visualization of surface temperature anomalies over the past century. The project uses **WebGL** technology to create a interactive 3D globe, allowing users to explore the data in a engaging and immersive way. The data used in this project is taken from the **NASA GISS Surface Temperature Analysis (GISTEMP v4)** dataset, which provides a comprehensive record of surface air temperature measurements from around the world.

## Features
The Climate Change Visualization project includes several key features, including:
* Interactive 3D globe visualization of surface temperature anomalies
* Data from **NASA GISS Surface Temperature Analysis (GISTEMP v4)** dataset
* Adaptive mobile-friendly website layout
* Smoothing of temperature data using a 250km radius
* Option to download **NetCDF** files from the [GISTEMP v4 website](https://data.giss.nasa.gov/gistemp/)

## Installation
To run the Climate Change Visualization project, users can visit the [live version of the project](http://students.washington.edu/aodhan/webgl_globe.html). The project can also be downloaded from the [GitHub repository](https://github.com/AodhanSweeney/climate_change_globe) and run locally.

### Data Sources
The project uses data from the **NASA GISS Surface Temperature Analysis (GISTEMP v4)** dataset, which can be downloaded from the [GISTEMP v4 website](https://data.giss.nasa.gov/gistemp/). The data is provided in **NetCDF** format and can be used for further analysis and visualization.
