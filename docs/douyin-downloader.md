Douyin Downloader is a **cross-platform** tool designed to download **Douyin videos** with ease. Built using **Tauri**, a framework for building desktop applications, this tool provides a simple and intuitive way to save Douyin videos on various operating systems, including **Windows**, **MacOS**, and **Linux**.

## Overview
The Douyin Downloader is a user-friendly application that allows users to download Douyin videos quickly and efficiently. With its **cross-platform** compatibility, users can access the tool on different devices, making it a convenient option for those who want to save their favorite Douyin videos.

## Features
The Douyin Downloader boasts several key features, including:
* **Cross-platform support**: The tool is compatible with **Windows**, **MacOS**, and **Linux** operating systems, allowing users to access it on their preferred device.
* **Easy-to-use interface**: The application has a simple and intuitive interface, making it easy for users to navigate and download Douyin videos.
* **Support for various video formats**: The Douyin Downloader supports different video formats, ensuring that users can download videos in their preferred format.

## Installation
To install the Douyin Downloader, users can follow these steps:
1. Download the application from the [release](https://github.com/lzdyes/douyin-downloader/releases) page.
2. Supports **Windows x64**, **Linux x86_64**, and **macOS 10.13+**.
Alternatively, users can build the application themselves, following the instructions provided in the repository.

### Notes
* If the application fails to start on **Windows**, users should ensure that they have **Webview2** installed on their system. The **Webview2** installation package can be downloaded from the [Microsoft Edge Webview2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/#download-section) website.
