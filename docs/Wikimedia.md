## Overview
The Wikimedia repository is a static site that renders MediaWiki markdown, also known as Wikitext, on the fly. It features a "blueboop" theme, inspired by [blueboop.is-a.dev](https://blueboop.is-a.dev), and allows users to create and access Wikipedia-like articles.

## Features
The repository has the following key features:
* **Static Site**: The site is static, meaning that it does not require a database or server-side processing to render content.
* **Wikitext Support**: The site supports MediaWiki markdown, allowing users to create formatted articles using Wikitext syntax.
* **Dynamic Rendering**: Articles are rendered on the fly, allowing users to access them via a simple URL.
* **Custom Theme**: The site features a custom "blueboop" theme, giving it a unique and distinctive look.

## Installation
To use the Wikimedia repository, follow these steps:
1. Clone the repository to your local machine.
2. Create a new `.mw` file in the `content/` folder (e.g., `My_Article.mw`).
3. Use Wikitext syntax for formatting your article.
4. Access the article via `index.html#/My_Article`.

## Usage
To add content to the repository, follow these steps:
1. Create a new `.mw` file in the `content/` folder.
2. Use Wikitext syntax for formatting.
3. Access the article via `index.html#/Article_Name`.
The repository uses a simple and intuitive system for creating and accessing articles, making it easy for users to contribute and share content.

## Theme Inspiration
The "blueboop" theme was inspired by [blueboop.is-a.dev](https://blueboop.is-a.dev), a website that features a similar aesthetic. The theme gives the repository a unique and distinctive look, setting it apart from other Wikipedia-like sites.
