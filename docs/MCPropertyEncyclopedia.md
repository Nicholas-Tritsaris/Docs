The MCPropertyEncyclopedia is a comprehensive online tool for viewing, filtering, and sorting all blocks, entities, and items by their properties in Minecraft. It is a powerful resource for players, developers, and content creators alike, providing a vast repository of information about the game's components. The tool is available online at [https://JoakimThorsen.github.io/MCPropertyEncyclopedia](https://JoakimThorsen.github.io/MCPropertyEncyclopedia) and is maintained by a small group of volunteers.

## Overview
The MCPropertyEncyclopedia is an ongoing project that aims to catalog and provide access to the properties of all blocks, entities, and items in Minecraft. The project relies on both automated and manual curation of data, which may lead to some information being incomplete or incorrect. However, the community is encouraged to contribute to the project through pull requests, issue reports, and discussions on the dedicated [Discord server](https://discord.gg/RUEVmTahYg).

## Features
The MCPropertyEncyclopedia offers several key features that make it a valuable resource for Minecraft enthusiasts:
* **Property selection**: Users can select from a wide range of properties to view and filter the data.
* **Filtering**: The tool allows users to narrow down the list of entries based on specific property values.
* **Sorting**: Users can sort the list by any property, including numerical and text values.
* **Search**: A basic case-insensitive search feature is available, enabling users to quickly find specific information.

## Usage
To get the most out of the MCPropertyEncyclopedia, users can follow these steps:
1. Select the desired properties to view and filter the data.
2. Use the filtering feature to narrow down the list of entries based on specific property values.
3. Sort the list by any property to organize the data in a meaningful way.
4. Utilize the search bar to quickly find specific information.

## Community Involvement
The MCPropertyEncyclopedia is a community-driven project, and contributions are welcome in various forms:
* **Pull requests**: Users can submit pull requests to add new data, correct existing information, or improve the tool's functionality.
* **Issue reports**: Users can report issues or suggest new features on the project's issue tracker.
* **Discord server**: The dedicated Discord server is the primary hub for discussions, updates, and community engagement.
* **Contributors**: A full list of contributors can be found on the [About Page](https://JoakimThorsen.github.io/MCPropertyEncyclopedia/about.html).

## Notes
As the MCPropertyEncyclopedia is an ongoing project, some information may be incomplete or incorrect. However, the community's contributions and feedback are essential to improving the tool's accuracy and usefulness. By working together, the Minecraft community can create a comprehensive and reliable resource for exploring the game's properties.
