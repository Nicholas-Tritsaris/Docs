# SamRewritten

SamRewritten is a modern **Steam achievements manager** for **Windows** and **Linux**, allowing users to unlock and relock achievements on their **Steam** account, as well as edit stats for games and apps that expose them. This project is heavily inspired by other notable projects, including **Steam Achievements Manager** by Gibbed and **Samira** by jsnli. With its user-friendly interface and robust features, SamRewritten provides a convenient way for users to manage their **Steam** achievements.

### Overview

SamRewritten is a tool designed to provide users with control over their **Steam** achievements, enabling them to unlock and relock achievements, as well as edit stats for supported games and apps. The project is not affiliated with **Valve Corporation** or **Microsoft**, and is instead a community-driven initiative.

### Features

The key features of SamRewritten include:

* Unlocking and relocking **Steam** achievements
* Editing stats for games and apps that expose them
* Support for both **Windows** and **Linux** operating systems
* A user-friendly interface, available in multiple themes, including **GTK** and **Adwaita**

### Installation

To install SamRewritten, users can download the latest release from the [GitHub releases page](https://github.com/PaulCombal/SamRewritten/releases). For **Windows**, an installer is available, while **Linux** users can download an **AppImage**. Detailed installation instructions for **Windows** are available, including the recommendation to use the official installer.
