# basic computer games

### Summary

The **basic-computer-games** repository is an updated version of the classic _BASIC Computer Games_ book, featuring well-written examples in a variety of common **memory safe** and **general purpose scripting languages**. This project aims to modernize the first million-selling computer book for 2022 and beyond, making it accessible to a new generation of programmers.

### Overview

The original _BASIC Computer Games_ book was a groundbreaking publication that introduced many people to the world of computer programming. The **basic-computer-games** repository takes the original book's source code and updates it to run on modern systems, using a range of popular programming languages. The project includes games from the original book, each with its own subfolder containing implementations in different languages.

### Features

The repository includes the following key features:

* Updated versions of the original _BASIC Computer Games_ in multiple programming languages, including **C#**, **Java**, **JavaScript**, **Kotlin**, **Lua**, **Perl**, **Python**, **Ruby**, **Rust**, and **VB.NET**
* Each game has its own subfolder, making it easy to navigate and compare implementations across different languages
* The use of **memory safe** and **general purpose scripting languages** ensures that the code is secure and easy to work with
* The project is based on the [2022 TIOBE index of top languages](https://www.tiobe.com/tiobe-index/), ensuring that the chosen languages are popular and relevant

### Discussion and Community

The project has an active [discussion forum](https://discourse.codinghorror.com/t/-/7927) where contributors can share their work, ask questions, and collaborate with others. The forum includes a worklog and conversation around the project, making it easy to get involved and stay up-to-date with the latest developments.

### Project Structure

The repository is organized into folders, each containing the source code for a specific game. The original **BASIC** source code has been moved into a separate folder for each project, and subfolders have been created for each language implementation. The project structure is based on the [2022 TIOBE index of top languages](https://www.tiobe.com/tiobe-index/), with a focus on **memory safe** and **general purpose scripting languages**.

### Installation and Usage

To get started with the **basic-computer-games** repository, simply clone the repository and navigate to the folder for the game you're interested in. From there, you can explore the different language implementations and start playing with the code. You can also [read the original book](https://archive.org/details/basiccomputergam0000unse) or [play the original games in your browser](https://troypress.com/wp-content/uploads/user/js-basic/index.html) to get a feel for the project's roots.
