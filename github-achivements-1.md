# github achivements

## Summary

The github-achivements repository is a dedicated collection of GitHub profile badges and achievements, written in multiple languages. It aims to provide a comprehensive list of achievements that developers can earn on the GitHub platform, celebrating their growth and contributions to open source projects.

## Overview

The repository was created in response to GitHub's introduction of the Achievements section, which was announced following the launch of the first Mars Helicopter, [Ingenuity](https://en.wikipedia.org/wiki/Ingenuity_\(helicopter\)). The Achievements section includes badges for various accomplishments, such as the Mars 2020 Helicopter Mission badge, the [Arctic Code Vault](https://archiveprogram.github.com/arctic-vault/) badge, and a badge for sponsoring open source work via [GitHub Sponsors](https://github.com/sponsors).

### History of Achievements

On **July 9, 2022**, GitHub announced more Achievements, recognizing the many stages of a developer's coding journey. However, the specific criteria for these achievements were not provided. The github-achivements repository aims to fill this gap by providing a comprehensive list of available achievements.

## Features

The repository provides a full list of GitHub profile badges and achievements, written in multiple languages, including:

* [EN-US](file:///)
* [PT-BR](/broken/pages/5563226296bb7d1b2ab80a5fc6a50769813940b6) The repository is available in beta and is a new way for developers to celebrate their growth and contributions to open source projects on the GitHub platform.

## Installation

There is no installation required to use the github-achivements repository. It is a collection of markdown files that can be accessed directly on GitHub.

## Contribution

If you notice any achievements that are not listed or are incorrect, please open an issue on the repository. This will help to ensure that the list of achievements remains comprehensive and up-to-date.
