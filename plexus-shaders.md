# plexus shaders

Plexus-Shaders is a **WebGL** Chrome Experiment that utilizes **ThreeJS**, **Fragment Shaders**, and **Vertex Shaders** to create a stress-test music visualizer for modern graphics cards. This tech demo is designed to push the limits of **ThreeJS** technology on modern video-cards, featuring a particle system and post-processing shaders. The project is best experienced on **Google Chrome**, although it is also compatible with **Firefox**.

### Overview

The Plexus-Shaders project is a showcase of **WebGL** capabilities, leveraging the power of **ThreeJS** to generate a visually stunning music visualizer. The demo includes a particle system and various post-processing shaders, all of which can be run simultaneously at 60 frames per second on capable hardware.

### Features

* **Music Visualizer**: A dynamic visualization that responds to music, creating a unique and immersive experience.
* **Particle System**: A complex system of particles that interact with the music visualization, adding an extra layer of depth and visual interest.
* **Post-Processing Shaders**: A range of shaders that enhance the visual quality of the demo, including effects such as glow, blur, and distortion.
* **WebGL Compatibility**: The project is designed to work seamlessly with **WebGL**, ensuring a smooth and efficient experience on supported browsers.

### Installation

To experience the Plexus-Shaders demo, simply visit the [Live Demo](http://mgz.me/plexus-shaders) in a compatible browser, such as **Google Chrome** or **Firefox**. No installation is required, as the demo runs entirely in the browser.

#### System Requirements

* A modern graphics card with **WebGL** support
* A compatible browser, such as **Google Chrome** or **Firefox**
* A stable internet connection

#### Example Use Cases

* Stress-testing modern graphics cards with a demanding **WebGL** application
* Showcasing the capabilities of **ThreeJS** and **WebGL** in a real-world scenario
* Creating immersive and interactive visualizations for music and other forms of media
