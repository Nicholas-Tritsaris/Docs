# workspace

Douyin Downloader is a _cross-platform_ download tool built with **Tauri**, a framework for building desktop applications. It allows users to download videos from Douyin, a popular social media platform. The application is available for **Windows**, **MacOS**, and **Linux** operating systems.

### Overview

The Douyin Downloader is a user-friendly tool that provides a simple and efficient way to download Douyin videos. It is built using **Tauri**, a framework that enables developers to build desktop applications using web technologies. The application is designed to be _cross-platform_, making it accessible to a wide range of users.

### Features

The Douyin Downloader has several key features, including:

* **Cross-platform support**: The application is available for **Windows**, **MacOS**, and **Linux** operating systems.
* **Easy to use**: The application has a simple and intuitive interface, making it easy for users to download Douyin videos.
* **Support for multiple platforms**: The application allows users to download videos from Douyin, a popular social media platform.

### Installation

To install the Douyin Downloader, users can download the application from the [release](https://github.com/lzdyes/douyin-downloader/releases) page. The application supports **Windows x64**, **Linux x86\_64**, and **macOS 10.13+**. Alternatively, users can build the application themselves, following the instructions provided in the repository.

#### System Requirements

To run the Douyin Downloader on **Windows**, users must have **Webview2** installed. This is a prerequisite for the application to function correctly.
