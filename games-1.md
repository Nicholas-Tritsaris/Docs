# games

A collection of **HTML5 Games** is a repository that hosts a variety of old style addictive games written in _JavaScript_, _HTML_, and _CSS_. The games are available to play online, providing an entertaining experience for users. This repository is a great resource for those looking to explore the world of HTML5 game development or simply to have fun playing classic games.

### Overview

The **games** repository is a compilation of games that showcase the capabilities of HTML5 game development. The games are built using _JavaScript_, _HTML_, and _CSS_, making them accessible on a wide range of devices. The repository provides a unique opportunity for developers to learn from and contribute to the development of HTML5 games.

### Features

The repository features a collection of games that are:

* Written in **JavaScript**, **HTML**, and **CSS**
* Accessible online through a dedicated website
* Addictive and fun to play
* A great resource for developers looking to learn from and contribute to HTML5 game development

### Play

To play the games, users can visit the [games website](http://tommalbran.github.io/games/) and start playing immediately. The website provides a simple and easy-to-use interface, allowing users to navigate and play the games with ease.

### Installation

There is no installation required to play the games, as they are available online. However, developers who wish to contribute to the repository or run the games locally can clone the repository and follow the instructions provided in the repository's documentation.

### Contribution

The **games** repository is open to contributions from developers who wish to add new games or improve existing ones. Developers can fork the repository, make changes, and submit a pull request to have their changes reviewed and merged. This collaborative approach allows the repository to grow and improve over time, providing a wider range of games for users to enjoy.
