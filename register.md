# register

### Summary

The **register** repository on GitHub is a service that allows developers to obtain a **.is-a.dev** subdomain for their personal websites. This service provides a unique opportunity for developers to create a professional-looking online presence.

### Overview

The **is-a.dev** service is designed to be easy to use, with a simple registration process. Developers can fork the repository, follow the instructions on the [documentation](https://docs.is-a.dev/) page, and open a pull request to register their subdomain. The service is managed through a GitHub repository, where users can submit pull requests to register their desired subdomain.

### Features

* **Customizable subdomains**: Developers can choose their own subdomain name, allowing for a personalized online presence.
* **Easy registration process**: The registration process is straightforward, with clear instructions provided on the [documentation](https://docs.is-a.dev/) page.
* **Community support**: The **is-a.dev** community is active on [Discord](https://discord.gg/is-a-dev-830872854677422150), where users can ask questions, receive updates, and get support.

### Installation

To register for a **.is-a.dev** subdomain, follow these steps:

1. Fork the [register](https://github.com/is-a-dev/register) repository.
2. Follow the instructions on the [documentation](https://docs.is-a.dev/) page.
3. Open a pull request to register your subdomain.
4. Wait for your pull request to be reviewed and merged.
5. Once your pull request is merged, your DNS records will be published.
